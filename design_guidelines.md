# Дизайн-руководство для платформы ResCrub

## Подход к дизайну
**Reference-Based Approach**: Следуем дизайн-системе Cal.com с адаптацией для российского рынка защиты данных. Cal.com демонстрирует идеальный баланс профессионализма, доступности и современности, что критически важно для доверия пользователей к сервису защиты персональных данных.

## Ключевые принципы
- **Доверие через простоту**: Чистый, минималистичный интерфейс как у Cal.com
- **Профессионализм**: Серьезный подход к вопросам безопасности данных
- **Русская локализация**: Полная адаптация для российской аудитории

- **Фон (темная тема)**: 224 71% 4%
- **Текст (светлая тема)**: 224 71% 4%

### Акцентные цвета (используются умеренно)
- **Безопасность**: 142 71% 45% (зеленый для индикаторов защиты)
- **Прогресс**: 217 91% 60% (синий для прогресс-баров)

## Типографика
- **Основной шрифт**: Inter (Google Fonts) - как у Cal.com
- **Размеры**: text-sm (14px), text-base (16px), text-lg (18px), text-xl (20px)
- **Заголовки**: font-semibold до font-bold
- **Основной текст**: font-normal

## Система отступов
Используем Tailwind spacing units: **4, 6, 8, 12, 16**
- Мелкие элементы: p-4, m-6
- Основные компоненты: p-8, m-8
- Крупные секции: p-12, m-16

## Библиотека компонентов

### Навигация
- Чистая горизонтальная навигация как у Cal.com
- Боковое меню для личного кабинета с иконками Heroicons

### Формы
- Закругленные поля ввода (rounded-lg)
- Фокусное состояние с border-primary
- Валидация с цветовой индикацией

### Дашборд
- Карточки с shadow-sm и rounded-xl
- Индикаторы прогресса с градиентной заливкой
- Статистические виджеты в стиле Cal.com

### Кнопки
- Первичные: bg-primary с rounded-lg
- Вторичные: outline стиль
- На изображениях: variant="outline" с размытым фоном

## Структура страниц

### Главная страница (marketing)
- **Минималистичный подход**: максимум 4 секции
- **Герой**: Крупный заголовок + подзаголовок + CTA кнопка
- **Преимущества**: 3-4 ключевых пункта с иконками
- **Тарифы**: Простая таблица в стиле Cal.com
- **CTA**: Финальный призыв к действию

### Личный кабинет (/app)
- **Дашборд**: Обзор статуса защиты данных
- **Мониторинг**: Детальные отчеты и прогресс
- **Профили**: Управление данными пользователей
- **Настройки**: Конфиденциальность и уведомления

## Изображения
- **Герой-изображение**: Современная иллюстрация защиты данных (в стиле Cal.com - чистая, минималистичная)
- **Иконки процессов**: Простые линейные иконки для этапов защиты данных
- **Статистические элементы**: Графики и диаграммы в фирменных цветах

## Анимации
Минимальные, деликатные анимации:
- Переходы между состояниями: transition-all duration-200
- Hover эффекты для кнопок и карточек
- Плавная загрузка контента

## Адаптивность
- Mobile-first подход
- Breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px)
- Гибкая типографика и отступы для всех устройств

## Темная тема
Полная поддержка темной темы с автоматическим переключением и сохранением предпочтений пользователя. Консистентная реализация во всех компонентах приложения.
